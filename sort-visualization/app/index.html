Documents

================

The visulalization including 5 different sorting algorithms: Bubble, Selection, Insertion, Merge and Quick Sort

There are 2 different kind of data, the first is pretty dataset, including an integer dataset from 1 to n. Other is a normal dataset including different float number.

Besides, you can switch to different algorithm, running speed, and dataset as you like.

Clearly, the bigger dataset is, the faster speed MergeSort and QuickSort shows.


Reference

================

1. Using gulp, Sinon, chai as dev tools to build the whole project.

2. Using mocha as test tools to run tests.

3. Using Html 5 Canvas to draw the bar graph.
